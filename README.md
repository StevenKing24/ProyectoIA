Título del Proyecto

Confianza en los agentes de inteligencia artificial

Estudiante

Steven Rey Maldonado

Objetivo del Proyecto

El objetivo de este proyecto es realizar el análisis, procesamiento y modelado de un dataset utilizando técnicas de Machine Learning y análisis de datos.

Se busca:

Explorar y comprender la estructura del conjunto de datos.
Realizar limpieza y preprocesamiento de la información.
Analizar patrones y comportamientos presentes en los datos.
Implementar modelos de aprendizaje automático para clasificación o predicción.
Evaluar el rendimiento del modelo mediante métricas de desempeño.

Este proyecto permite aplicar conceptos de ciencia de datos, estadística y aprendizaje automático en un caso práctico.

Información del Dataset
Descripción General

El dataset contiene información utilizada para el entrenamiento y evaluación de modelos de Machine Learning. Los datos incluyen diferentes variables o características que representan las columnas que muestran como los usuarios confian en las respuestas que arroja los distintos agentes de inteligencia artificial.

Contenido del Dataset

El conjunto de datos incluye: 

Variables numéricas (porcentajes de confianza o porcentajes de acierto)
Variables categóricas (tipo de consulta o familiarizacion con las tecnlogias)
